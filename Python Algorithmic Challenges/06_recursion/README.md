# 📘 Recursion — Теория

## Что такое рекурсия?

Рекурсия — когда функция **вызывает сама себя**.

```python
def factorial(n):
    if n <= 1:         # базовый случай
        return 1
    return n * factorial(n - 1)  # рекурсивный вызов
```

## Два обязательных компонента

1. **Базовый случай** — условие остановки
2. **Рекурсивный случай** — вызов себя с уменьшением задачи

## Стек вызовов

Каждый вызов добавляется в стек. Глубина ограничена (~1000 в Python).
```python
import sys
sys.setrecursionlimit(10000)  # увеличить лимит
```

## Мемоизация

```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
    if n < 2: return n
    return fib(n-1) + fib(n-2)
```

## Виды рекурсии
- **Линейная** → один рекурсивный вызов
- **Двоичная (binary)** → два вызова (деревья, Фибоначчи)
- **Хвостовая (tail)** → рекурсивный вызов — последняя операция
- **Backtracking** → перебор с откатом

## Паттерны
- **Разделяй и властвуй** → разбить → решить → объединить
- **Backtracking** → попробовать → откатить если не подходит
- **Дерево решений** → перебор вариантов
