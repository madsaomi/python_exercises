## Математические паттерны

### Модульная арифметика


### Степени двойки
```python
def isPowerOfTwo(n):
    return n > 0 and (n & (n - 1)) == 0
```

### НОД и НОК
```python
import math
math.gcd(a, b)
math.lcm(a, b)  # Python 3.9+
```

### Проверка простого числа
```python
def isPrime(n):
    if n < 2: return False
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0: return False
    return True
```

## Типичные паттерны циклов
