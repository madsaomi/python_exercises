## 📚 Подробное описание тем

### 01 — Массивы и списки
Работа с индексами и срезами, двойные указатели (two pointers), скользящее окно (sliding window), поиск подмассивов и подпоследовательностей, операции на месте (in-place).  
**Паттерны:** Two Pointers, Sliding Window, Prefix Sum  
**На собеседованиях:** Two Sum, Maximum Subarray, Rotate Array

### 02 — Строки
Обработка символов и подстрок, палиндромы и анаграммы, скользящее окно для строк, кодирование и декодирование.  
**Паттерны:** Two Pointers, Sliding Window, Hash Map для подсчёта  
**На собеседованиях:** Valid Anagram, Longest Substring Without Repeating Characters

### 03 — Хеш-таблицы и словари
Принципы работы хеш-таблиц, подсчёт частот и группировка, поиск пар с заданным свойством, кэширование.  
**Паттерны:** Frequency Counter, Hash Set, Two Sum Pattern  
**На собеседованиях:** Group Anagrams, Top K Frequent Elements

### 04 — Циклы и математика
Числовые алгоритмы (GCD, LCM, простые числа), битовые операции, модульная арифметика, обработка цифр числа.  
**Паттерны:** Math Tricks, Bit Manipulation, Modular Arithmetic  
**На собеседованиях:** Palindrome Number, Power of Two, Count Primes

### 05 — Функции
Функции высшего порядка (map, filter, reduce), замыкания, лямбда-функции, декомпозиция задач.  
**Паттерны:** Divide and Conquer, Callback Pattern  
**На собеседованиях:** Function Composition, Memoize Function

### 06 — Рекурсия
Базовый и рекурсивный случаи, backtracking, генерация перестановок и комбинаций, рекурсивный обход структур.  
**Паттерны:** Backtracking, Divide and Conquer, Memoization  
**На собеседованиях:** Generate Parentheses, Permutations, Subsets, N-Queens

### 07 — Сортировка
Алгоритмы: Bubble, Selection, Insertion, Merge, Quick. Сложность и стабильность, кастомные компараторы.  
**Паттерны:** Merge Sort, Quick Sort, Counting Sort  
**На собеседованиях:** Merge Intervals, Sort Colors, Kth Largest Element

### 08 — Бинарный поиск
Классический бинарный поиск, поиск границ, бинарный поиск по ответу, поиск в повёрнутом массиве.  
**Паттерны:** Binary Search on Index, Binary Search on Answer  
**На собеседованиях:** Search in Rotated Sorted Array, Find Peak Element

### 09 — Стек и очередь
Реализация стека и очереди, монотонный стек, проверка скобочных последовательностей, BFS с очередью, deque.  
**Паттерны:** Monotonic Stack, BFS with Queue, Min Stack  
**На собеседованиях:** Valid Parentheses, Min Stack, Daily Temperatures

### 10 — Связные списки
Односвязные и двусвязные списки, быстрый и медленный указатели, реверс списка, обнаружение цикла.  
**Паттерны:** Fast & Slow Pointers, Dummy Node, In-place Reversal  
**На собеседованиях:** Reverse Linked List, Merge Two Sorted Lists, Linked List Cycle

### 11 — Деревья
Бинарные деревья и BST, обходы (inorder, preorder, postorder, level-order), поиск LCA, валидация BST.  
**Паттерны:** DFS (Recursive), BFS (Level Order), Tree Construction  
**На собеседованиях:** Maximum Depth, Invert Binary Tree, Validate BST

### 12 — Динамическое программирование
Мемоизация «сверху вниз», табуляция «снизу вверх», 1D и 2D DP, задачи на подпоследовательности и рюкзак.  
**Паттерны:** Memoization, Tabulation, State Transition  
**На собеседованиях:** Climbing Stairs, Longest Common Subsequence, Coin Change

## 📊 Ключевые паттерны

| Паттерн | Описание | Где встречается |
|---------|----------|-----------------|
| **Two Pointers** | Два указателя навстречу или в одном направлении | Arrays, Strings, Linked Lists |
| **Sliding Window** | Окно фиксированного/переменного размера | Arrays, Strings |
| **Hash Map** | Быстрый поиск и подсчёт через словарь | Hash Maps, Arrays, Strings |
| **Binary Search** | Деление пространства поиска пополам | Binary Search, Sorting |
| **DFS / BFS** | Обход в глубину / ширину | Trees, Recursion, Stack & Queue |

| **Dynamic Programming** | Разбиение на подзадачи с мемоизацией | DP, Recursion |

## ⏱️ Шпаргалка по Big O

| Сложность | Название | Пример | n = 10⁶ |
|-----------|----------|--------|---------|
| O(1) | Константная | Доступ по индексу | ✅ мгновенно |
| O(log n) | Логарифмическая | Бинарный поиск | ✅ ~20 оп. |
| O(n) | Линейная | Один проход | ✅ 10⁶ оп. |
| O(n log n) | Линейно-логарифмическая | Merge Sort | ✅ ~2·10⁷ |
| O(n²) | Квадратичная | Два вложенных цикла | ⚠️ медленно |
| O(2ⁿ) | Экспоненциальная | Перебор подмножеств | ❌ нереально |
