## 🚀 Как пользоваться

1. Откройте папку нужной темы
2. Прочитайте `README.md` — краткая теория
3. Откройте `tasks.md` — прочитайте условие задачи
4. Напишите решение в `exercises.py`
5. Протестируйте локально

## 🎯 Уровни сложности

- 🟢 **Easy** — базовые задачи, 1 подход
- 🟡 **Medium** — требуют знания алгоритмов
- 🔴 **Hard** — сложные, комбинация подходов
