## Задача 1: FizzBuzz (5 мин)

Напишите функцию `fizzbuzz(n)`, которая возвращает список строк от 1 до n:
- Если число делится на 3 → "Fizz"
- Если на 5 → "Buzz"
- Если на 3 и 5 → "FizzBuzz"
- Иначе → строковое представление числа

```python
fizzbuzz(15)[-5:]  # → ["11", "Fizz", "13", "14", "FizzBuzz"]
```

<details>
<summary>🔍 Подвох</summary>

Проверяйте делимость на 15 **первым**! Частая ошибка:
```python
# ❌ Неправильный порядок
if i % 3 == 0: "Fizz"
elif i % 5 == 0: "Buzz"
elif i % 15 == 0: "FizzBuzz"  # никогда не достигнется!
```
</details>

## Задача 2: Two Sum (10 мин)

Найдите два числа в массиве, сумма которых равна `target`. Верните их индексы.

```python
two_sum([2, 7, 11, 15], 9)  # → [0, 1]
two_sum([3, 2, 4], 6)       # → [1, 2]
```

<details>
<summary>🔍 Подвох</summary>

Наивное решение O(n²) — два цикла. Оптимальное O(n) — hash map:
```python
# Не забудьте: нельзя использовать один элемент дважды!
# [3, 3], target=6 → [0, 1] (разные индексы)
```
</details>

## Задача 3: Valid Parentheses (10 мин)

Проверьте, правильная ли скобочная последовательность.

```python
is_valid("()[]{}")     # → True
is_valid("(]")         # → False
is_valid("([)]")       # → False
is_valid("{[]}")       # → True
```

<details>
<summary>🔍 Подвох</summary>

- Пустая строка — `True`
- Только закрывающие скобки `")"` — `False`
- Нечётная длина — `False` (можно сразу отсечь)
</details>

## Задача 5: Merge Intervals (15 мин)

Объедините пересекающиеся интервалы.


## Задача 7: Maximum Subarray (Kadane's) (10 мин)

Найдите подмассив с максимальной суммой.

```python
max_subarray([-2,1,-3,4,-1,2,1,-5,4])  # → 6 (подмассив [4,-1,2,1])
max_subarray([1])                        # → 1
max_subarray([-1,-2,-3])                 # → -1
```

<details>
<summary>🔍 Подвох</summary>

Алгоритм Кадане: `current_sum = max(num, current_sum + num)`. Не забывайте случай, когда **все числа отрицательные** — ответ не 0, а максимальное отрицательное!
</details>

## Задача 9: Most Common Word (10 мин)

```python

```

<details>
<summary>🔍 Подвох</summary>

- Приведите к нижнему регистру
- Удалите пунктуацию
- Отфильтруйте banned-слова
- Используйте `Counter.most_common()`
</details>
