# 📘 Тема 6: Кортежи и множества

## Кортежи (tuple)

**Кортеж** — неизменяемая упорядоченная коллекция.

```python
t = (1, 2, 3)
t2 = 1, 2, 3          # скобки необязательны
single = (42,)         # кортеж из одного элемента (запятая!)
empty = ()

t[0]     # 1
t[-1]    # 3
t[1:3]   # (2, 3)
len(t)   # 3
# t[0] = 10  # ❌ TypeError — кортежи неизменяемы
```

### Когда использовать кортежи?
- Координаты: `point = (3, 5)`
- Возврат нескольких значений из функции
- Ключи в словарях (списки нельзя)
- Данные, которые не должны меняться

### Распаковка

```python
x, y, z = (1, 2, 3)
first, *rest = (1, 2, 3, 4, 5)

# Именованные кортежи
from collections import namedtuple
Point = namedtuple('Point', ['x', 'y'])
p = Point(3, 5)
print(p.x, p.y)  # 3 5
```

## Множества (set)

**Множество** — неупорядоченная коллекция **уникальных** элементов.

```python
s = {1, 2, 3, 4, 5}
s2 = set([1, 2, 2, 3, 3])  # {1, 2, 3}
empty = set()               # НЕ {}, это словарь!
```

### Методы множеств

```python
s = {1, 2, 3}
s.add(4)           # {1, 2, 3, 4}
s.remove(2)        # {1, 3, 4} — ошибка если нет
s.discard(10)      # ничего — нет ошибки
s.pop()            # удалить случайный элемент
s.clear()          # очистить
```

### Операции над множествами

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b   # объединение: {1,2,3,4,5,6}
a & b   # пересечение: {3,4}
a - b   # разность: {1,2}
a ^ b   # симметричная разность: {1,2,5,6}
a <= b  # подмножество?
```

### Frozenset — неизменяемое множество

```python
fs = frozenset([1, 2, 3])
# fs.add(4)  # ❌ AttributeError
```
