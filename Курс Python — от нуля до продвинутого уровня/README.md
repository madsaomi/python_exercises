# 🐍 Курс Python — от нуля до продвинутого уровня

Полный курс по Python с теорией и практическими заданиями.  
Каждая тема содержит теоретический материал и **10 упражнений** для закрепления.

> Этот курс охватывает **20 тем** и содержит **200 упражнений** — от самых основ до продвинутых концепций.  
> Идеально подходит для самостоятельного изучения Python с нуля или для систематизации уже имеющихся знаний.

## 📚 Подробное описание тем

### Блок 1 — Основы Python (темы 01–05)

**01 · Переменные и типы данных**  
Знакомство с Python: переменные, числа (int, float), строки, булевы значения, None. Преобразование типов, функции type() и isinstance(). Правила именования переменных.

**02 · Строки и их методы**  
Создание и форматирование строк (f-strings, format). Срезы, индексация. Методы: upper(), lower(), split(), join(), replace(), find(), strip(). Экранирование и raw-строки.

**03 · Условные конструкции**  
if / elif / else, тернарный оператор, логические операторы (and, or, not). Вложенные условия, match-case (Python 3.10+). Truthiness и falsy-значения.

**04 · Циклы**  
Цикл for (range, enumerate, zip), цикл while. break, continue, else в циклах. Вложенные циклы, итерация по строкам и коллекциям.

**05 · Списки**  
Создание, индексация, срезы. Методы: append(), extend(), insert(), remove(), pop(), sort(), reverse(). Копирование списков (shallow vs deep copy). Списки списков.

---

### Блок 2 — Структуры данных и функции (темы 06–09)

**06 · Кортежи и множества**  
Кортежи: неизменяемость, распаковка, именованные кортежи (namedtuple). Множества: операции (union, intersection, difference), frozenset. Когда что использовать.

**07 · Словари**  
Создание, доступ, методы: get(), keys(), values(), items(), setdefault(), update(). defaultdict, Counter, OrderedDict. Вложенные словари, итерация.

**08 · Функции**  
Определение функций, параметры и аргументы (*args, **kwargs). Возвращаемые значения, область видимости (LEGB). Лямбда-функции, map(), filter(), reduce(). Аннотации типов.

**09 · Работа с файлами**  
Чтение и запись файлов (open, read, write). Контекстный менеджер (with). Работа с CSV, JSON. Модуль pathlib. Кодировки (UTF-8). Обработка больших файлов.

---

### Блок 3 — ООП и обработка ошибок (темы 10–12)

**10 · Основы ООП**  
Классы и объекты, __init__, self. Атрибуты класса и экземпляра. Методы: обычные, статические (@staticmethod), классовые (@classmethod). Инкапсуляция, свойства (@property).

**11 · Продвинутое ООП**  
Наследование (одиночное, множественное, MRO). Полиморфизм, абстрактные классы (ABC). Магические методы (__str__, __repr__, __eq__, __lt__, __add__, __len__). Дескрипторы, метаклассы.

**12 · Обработка исключений**  
try / except / else / finally. Встроенные исключения (ValueError, TypeError, KeyError и др.). Создание собственных исключений. Цепочки исключений (raise from). Контекстные менеджеры.

---

### Блок 4 — Продвинутые концепции (темы 13–16)

**13 · Модули и пакеты**  
Импорт модулей (import, from ... import). Создание собственных модулей и пакетов. __init__.py, __name__ == "__main__". Стандартная библиотека: os, sys, math, random, datetime, collections.

**14 · Декораторы и генераторы**  
Декораторы: создание, применение, цепочки декораторов, functools.wraps. Генераторы: yield, генераторные выражения, itertools. Ленивые вычисления. Корутины (основы).

**15 · Comprehensions**  
List comprehension, dict comprehension, set comprehension. Вложенные comprehensions, условные выражения. Когда использовать, а когда — обычный цикл. Производительность.

**16 · Регулярные выражения**  
Модуль re: search(), match(), findall(), sub(), split(). Синтаксис: метасимволы, квантификаторы, группы, lookahead/lookbehind. Компиляция паттернов. Практические примеры.

---

### Блок 5 — Прикладные навыки (темы 17–20)

**17 · Базы данных (SQLite)**  
Модуль sqlite3: подключение, создание таблиц, CRUD-операции. Параметризованные запросы (защита от SQL-инъекций). Транзакции, индексы. ORM-подход (основы).

**18 · Работа с API**  
Модуль requests: GET, POST, PUT, DELETE. Работа с JSON. Аутентификация (API keys, OAuth). Обработка ошибок HTTP. Пагинация. Пример: работа с реальным REST API.

**19 · Тестирование**  
unittest: TestCase, assert-методы, setUp/tearDown. pytest: фикстуры, параметризация, маркеры. Моки (unittest.mock). TDD-подход. Покрытие кода (coverage).

**20 · Виртуальные окружения и pip**  
venv, virtualenv: создание и активация. pip: установка, обновление, удаление пакетов. requirements.txt, pip freeze. Альтернативы: pipenv, poetry. .env файлы.

## 🚀 Как пользоваться

### Быстрый старт

```bash
git clone <url>
cd python-course
code .                        # откройте в VS Code
cd 01_variables_and_types     # начните с первой темы
```

### Рабочий процесс

1. **Начните с папки** нужной темы (по порядку)
2. **Прочитайте** файл `theory.md` — подробная теория с примерами кода
3. **Откройте** файл `exercises.py` — выполните все 10 заданий
4. **Решения** пишите прямо в файле в отведённых местах
5. **Запустите** файл для проверки: `python exercises.py`
6. **Переходите** к следующей теме

### Пример

```bash
# Откройте теорию
cat 01_variables_and_types/theory.md

# Решайте задания
python 01_variables_and_types/exercises.py
```

**Сколько времени нужно на весь курс?**  
При ежедневных занятиях по 1–2 часа — **6–10 недель** на весь курс.

**Нужны ли предварительные знания?**
