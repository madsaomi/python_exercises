# 🐍 Курс Python — от нуля до продвинутого уровня

Полный курс по Python с теорией и практическими заданиями.  
Каждая тема содержит теоретический материал и **10 упражнений** для закрепления.

---

## 📂 Структура курса

| №  | Папка | Тема |
|----|-------|------|
| 01 | `01_variables_and_types` | Переменные и типы данных |
| 02 | `02_strings` | Строки и их методы |
| 03 | `03_conditions` | Условные конструкции |
| 04 | `04_loops` | Циклы (for, while) |
| 05 | `05_lists` | Списки |
| 06 | `06_tuples_and_sets` | Кортежи и множества |
| 07 | `07_dictionaries` | Словари |
| 08 | `08_functions` | Функции |
| 09 | `09_file_handling` | Работа с файлами |
| 10 | `10_oop_basics` | Основы ООП |
| 11 | `11_oop_advanced` | Продвинутое ООП |
| 12 | `12_exceptions` | Обработка исключений |
| 13 | `13_modules_and_packages` | Модули и пакеты |
| 14 | `14_decorators_and_generators` | Декораторы и генераторы |
| 15 | `15_comprehensions` | List/Dict/Set Comprehensions |
| 16 | `16_regular_expressions` | Регулярные выражения |
| 17 | `17_databases_sqlite` | Базы данных (SQLite) |
| 18 | `18_working_with_api` | Работа с API (requests) |
| 19 | `19_testing` | Тестирование (unittest, pytest) |
| 20 | `20_virtual_env_and_pip` | Виртуальные окружения и pip |

---

## 🚀 Как пользоваться

1. Начните с папки `01_variables_and_types`
2. Прочитайте файл `theory.md` — теория по теме
3. Откройте файл `exercises.py` — выполните все 10 заданий
4. Решения пишите прямо в файле в отведённых местах
5. Переходите к следующей теме

## 🎯 Требования

- Python 3.8+
- Текстовый редактор или IDE (VS Code, PyCharm)
- Желание учиться! 💪
