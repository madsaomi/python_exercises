# 📘 Тема 1: Переменные и типы данных

## Что такое переменная?

**Переменная** — это именованная область памяти, в которой хранится значение.  
В Python переменные создаются в момент присваивания — не нужно заранее объявлять тип.

```python
name = "Алиса"      # строка (str)
age = 25             # целое число (int)
height = 1.68        # дробное число (float)
is_student = True    # логическое значение (bool)
```

---

## Основные типы данных

| Тип | Описание | Пример |
|-----|----------|--------|
| `int` | Целое число | `42`, `-7`, `0` |
| `float` | Дробное число | `3.14`, `-0.5`, `1.0` |
| `str` | Строка | `"Привет"`, `'Python'` |
| `bool` | Логический тип | `True`, `False` |
| `NoneType` | Отсутствие значения | `None` |

---

## Функция `type()`

Чтобы узнать тип переменной, используйте `type()`:

```python
x = 10
print(type(x))  # <class 'int'>

y = "hello"
print(type(y))  # <class 'str'>
```

---

## Преобразование типов (casting)

Python позволяет явно преобразовывать типы данных:

```python
# Строка → число
num_str = "42"
num = int(num_str)      # 42
num_f = float(num_str)  # 42.0

# Число → строка
age = 25
age_str = str(age)      # "25"

# Число → логический тип
print(bool(0))    # False
print(bool(1))    # True
print(bool(-5))   # True
print(bool(""))   # False
print(bool("hi")) # True
```

## Правила именования переменных

✅ **Можно:**
- Использовать буквы (a-z, A-Z), цифры (0-9) и подчёркивание `_`
- Начинать с буквы или подчёркивания
- Использовать snake_case (рекомендуется)

❌ **Нельзя:**
- Начинать с цифры: `1name = "ошибка"`
- Использовать пробелы: `my name = "ошибка"`
- Использовать зарезервированные слова: `if`, `for`, `class` и т.д.

```python
# ✅ Правильно
user_name = "Алиса"
_private = 42
myAge = 25

# ❌ Неправильно
# 2name = "ошибка"
# my-name = "ошибка"
```

## Множественное присваивание

```python
# Присвоить разные значения нескольким переменным
a, b, c = 1, 2, 3

# Присвоить одно значение нескольким переменным
x = y = z = 0

# Обмен значений (swap)
a, b = b, a
```

## Ввод данных — `input()`

Функция `input()` **всегда** возвращает строку (`str`):

```python
name = input("Введите имя: ")
age = int(input("Введите возраст: "))  # Преобразуем в число
```

---

## Вывод данных — `print()`

```python
name = "Python"
version = 3.12

# Обычный вывод
print("Hello, World!")

# Несколько аргументов
print("Язык:", name, "Версия:", version)

# f-строки (рекомендуется)
print(f"Язык: {name}, версия: {version}")

# Параметр sep — разделитель
print("a", "b", "c", sep="-")  # a-b-c

# Параметр end — окончание строки
print("Hello", end=" ")
print("World")  # Hello World
```

## Арифметические операции

| Оператор | Описание | Пример | Результат |
|----------|----------|--------|-----------|
| `+` | Сложение | `5 + 3` | `8` |
| `-` | Вычитание | `5 - 3` | `2` |
| `*` | Умножение | `5 * 3` | `15` |
| `/` | Деление | `7 / 2` | `3.5` |
| `//` | Целочисленное деление | `7 // 2` | `3` |
| `%` | Остаток от деления | `7 % 2` | `1` |
| `**` | Возведение в степень | `2 ** 3` | `8` |

---

## Составные операторы присваивания

```python
x = 10
x += 5   # x = x + 5  → 15
x -= 3   # x = x - 3  → 12
x *= 2   # x = x * 2  → 24
x /= 4   # x = x / 4  → 6.0
x //= 2  # x = x // 2 → 3.0
x %= 2   # x = x % 2  → 1.0
x **= 3  # x = x ** 3 → 1.0
```
